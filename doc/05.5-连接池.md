# 5.5 连接池

连接池的重要性,这时就不赘述了,下面具体介绍框架中实现的哪些连接池。

有关于连接池有一个配置项

```
//启动一个新进程维护连接池
$config['asyn_process_enable'] = false;
```

## Redis连接池

### 主要特性

- 支持异步+协程
- 支持分布式自动分片
- 支持master-slave读写分离
- 支持故障自动failover
- 支持断线重连
- 支持自动提取和归还连接
