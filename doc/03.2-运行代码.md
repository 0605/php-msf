# 3.2 运行代码

这里以运行示例项目代码为例

## 克隆

```bash
$>git clone git@github.com:PGwireless/php-msf.git php-msf
$>cd php-msf
$>git checkout demo
```

## 运行

### build.sh(docker环境需要)

```bash
$>cd php-msf
$>./build.sh
```

### composer

```bash
$>cd /home/worker/data/www/php-msf
$>composer install -vvv
```

### supervisor

```bash
$>supervisorctl
supervisor> update
php-msf: added process group
supervisor> status
msf:php-msf            RUNNING   pid 26958, uptime 0:00:31
//...
```
