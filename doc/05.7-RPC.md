# 5.7 RPC

RPC（Remote Procedure Call）即远程过程调用,是指网络计算机之间相互调用过程函数,而不需要关心底层的网络通信。MSF的RPC 目前是基于Http协议,当然使用者不需要关心,因为后续有可能会有调整。

MSF实现的RPC具体的基本原则是:

1. 任何类可导出并提供服务
2. RPC Client与RPC Server的类名与方法名一致
3. 任何服务可以很方便的导出自己的类与类方法
