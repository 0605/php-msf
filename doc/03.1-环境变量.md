# 3.1 环境变量

添加系统环境变量，相应的环境配置如下：

```bash
vi /etc/profile
...
export MSF_ENV=...
```

#### docker

export MSF_ENV=docker

#### dev

export MSF_ENV=dev

#### qa

export MSF_ENV=qa

#### online

export MSF_ENV=product_en
